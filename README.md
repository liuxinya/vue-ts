# UNION-VUE-TYPESCRIPT-QUICK-STARTER

> use the use with typescript

## 安装

```bash
npm install # yarn install
```

## 使用

```
npm run dev
```

## 假数据

```
cd ./server

yarn install

npm start
```

## 介绍

这是将`Typescript`和`Vue`结合的脚手架，主要使用了`vue-property-decorator` 和 `vue-template-loader'`。

为了能够自己去实现`render`函数，这里还加入了对`jsx`的支持。
